# Projeto Esteira 
Projeto consiste na manipulação de dados em registradores do microcontrolador Atmega328p, os mesmos sendo usados para uma simulação simples de manufatura de um produto.
##
## Tecnologias usadas
1. Assembly;
2. Simulação no Proteus.
